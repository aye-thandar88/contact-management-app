# Contact Management App

# React + TypeScript + Vite + Tailwind project with a JSON Server backend.

## Getting Started

### Start the backend (JSON Server)

```bash
npm run serve
```

This will run `json-server` and start your mock API at:

```
http://localhost:4000
```

### Start the frontend (React + Vite)

```bash
npm run dev
```

This will run the frontend development server at:

```
http://localhost:3000
```

## Workflow

1. Open **two terminal tabs/windows**.
2. In the first terminal, run:
   ```bash
   npm run serve
   ```
   This keeps your JSON backend running.
3. In the second terminal, run:
   ```bash
   npm run dev
   ```
   This serves your React + TypeScript + Tailwind frontend.

Once both are running, the frontend will connect to the backend automatically.

```js
      // Optionally, add this for stylistic rules

        project: ["./tsconfig.node.json", "./tsconfig.app.json"],
        tsconfigRootDir: import.meta.dirname,

  },
]);
```
